# 仕様 - CLIオプション

<br/>

```shell
$ ./systemd-cd --help
Usage of ./systemd-cd:
      --dir.backup string                  Path to service backup files (default "/var/backups/systemd-cd/")
      --dir.binary string                  Path to service binary files (default "/usr/local/systemd-cd/bin/")
      --dir.etc string                     Path to service etc files (default "/usr/local/systemd-cd/etc/")
      --dir.opt string                     Path to service opt files (default "/usr/local/systemd-cd/opt/")
      --dir.src string                     Path to service source files (default "/usr/local/systemd-cd/src/")
      --dir.systemd-unit-env-file string   Path to systemd env files (default "/usr/local/systemd-cd/etc/default/")
      --dir.systemd-unit-file string       Path to systemd unit files. (default "/usr/local/lib/systemd/system/")
      --dir.var string                     Path to variable files (default "/var/lib/systemd-cd/")
  -f, --file.manifest strings              Manifeset file path
      --log.level string                   Only log messages with the given severity or above.
                                           One of: [panic, fatal, error, warn, info, debug, trace] (default "info")
      --log.report-caller                  Enable log report caller
      --log.timestamp                      Enable log timestamp.
      --pipeline.interval                  Interval of repository polling (second) (default 180)
```
